## Getting Started
```bash
pip install jupyter
```
Once you have Jupyter installed, you can clone this repository and run the notebooks using the following commands:

```bash
git clone https://github.com/eriosta/analytics-applications.git
cd analytics-applications
jupyter notebook
```
